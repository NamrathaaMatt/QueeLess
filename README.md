# QueueLess

QueueLess is a smart queue prediction and crowd optimization web application prototype designed to reduce waiting time at public places.

## Features
- Category-based place selection
- Dynamic queue prediction using service time modeling
- One-time check-in system
- Automatic queue progression
- Notification when crowd reduces
- Analytics dashboard

## Tech Stack
HTML, CSS, JavaScript (Prototype version)

## Future Scope
- Real-time database integration
- GPS-based location detection
- AI-based waiting time prediction
- QR-based entry tracking
